# Wafa Abbas Web

## Admin GitHub Sign-In

The admin area now includes a GitHub sign-in flow for Supabase authentication.

### Setup

1. Create a Supabase project.
2. Enable the GitHub OAuth provider in Supabase Auth.
3. Replace the placeholder values in [admin/login.html](admin/login.html) and [admin/dashboard.html](admin/dashboard.html) with your Supabase project URL and anon key.
4. Open [admin/login.html](admin/login.html) in a browser to sign in.
